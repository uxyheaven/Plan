# 演讲稿区

这里存放演讲主题、提纲、逐字稿和相关讨论稿。

## 推荐写作顺序

1. 先明确听众、时长、目标
2. 再写核心主张
3. 再搭结构
4. 最后扩成逐字稿

## 常见拆法

- 单文件：适合短分享或早期草稿
- 多文件目录：适合大型演讲、课程或系列分享

多文件目录可按下面拆：

- `brief.md`：任务设定
- `outline.md`：结构提纲
- `script.md`：逐字稿
- `notes.md`：素材、案例、金句

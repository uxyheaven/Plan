# AI 分析系统

这是一个独立的方案专题目录，用来集中存放 AI 分析系统相关设计资料。

## 文档分工

- [overview.md](overview.md)：适合快速讨论和对齐的大纲版方案
- [spec.md](spec.md)：完整设计规范
- [prompt.md](prompt.md)：LLM 提示词参考
- [key-points.md](key-points.md)：速查要点、术语和常见陷阱

## 建议阅读顺序

1. 先看 `overview.md`
2. 再看 `spec.md`
3. 需要讨论提示词时看 `prompt.md`
4. 需要快速回顾时看 `key-points.md`
